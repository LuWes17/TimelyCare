# TimelyCare App - Participant Installation Instructions

## What You'll Need
- Android smartphone (Android 8.0+ / API 26+)
- Wear OS smartwatch (Wear OS 3.0+ / API 30+)
- Both devices paired and connected

## Installation Steps

### 1. Download the App
- You will receive the TimelyCare APK file via email or shared drive link
- Download the APK file to your Android smartphone

### 2. Enable Installation from Unknown Sources
1. Go to **Settings** > **Security** (or **Privacy & Security**)
2. Enable **Install apps from unknown sources** or **Unknown sources**
3. If using Android 8.0+, you may need to allow installation for your browser/file manager specifically

### 3. Install the Mobile App
1. Open your file manager or downloads folder
2. Tap the **TimelyCare.apk** file
3. Follow the installation prompts
4. Tap **Install**

### 4. Automatic Watch App Installation
- Once you install the mobile app, the watch app will **automatically install** on your paired smartwatch
- This may take 2-5 minutes depending on your connection
- You'll see a notification on your watch when installation is complete

### 5. Open the Apps
1. **Mobile App**: Find "TimelyCare" in your app drawer and open it
2. **Watch App**: Look for "TimelyCare" on your smartwatch and open it

## Troubleshooting

### Watch App Not Installing Automatically?
1. Ensure your phone and watch are connected (check Wear OS app)
2. Open the **Wear OS** app on your phone
3. Go to **Advanced features** > **Install apps on watch**
4. Look for TimelyCare and tap **Install**

### Can't Install APK?
- Make sure "Install from unknown sources" is enabled
- Try using a different file manager app
- Contact the researcher for assistance

### App Won't Open?
- Restart both your phone and watch
- Check that both devices meet the minimum requirements
- Contact the researcher if problems persist
